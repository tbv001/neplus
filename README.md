# Nodegraph Editor+

An addon for Garry's Mod that adds a tool to allow the user to edit a map's nodegraph.

[Steam Workshop Page](https://steamcommunity.com/sharedfiles/filedetails/?id=3387089606)

## Dependencies

- [HintNode Loader](https://steamcommunity.com/sharedfiles/filedetails/?id=3432853023) (Not really a requirement for using the tool, but it is required if you want to load hint nodes generated by the tool.)

## Installation

Navigate to your Garry's Mod addons folder, and then clone the repository:
```bash
git clone https://github.com/tbv001/neplus.git
```
